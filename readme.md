# Proyecto 2 - Sistema de Inventario en JavaScript

Este proyecto es una aplicación básica en JavaScript que permite gestionar un inventario de productos utilizando arreglos y métodos como `map`, `filter`, `reduce`, `sort`, entre otros.

---

## Funcionalidades

El sistema permite:

- Mostrar todos los productos
- Ver productos con stock bajo (menos de 5)
- Ver productos agotados
- Generar reportes del inventario
- Buscar productos
- Calcular totales de inventario y ventas
- Identificar el producto más vendido

---

## Métodos utilizados

Se aplican conceptos importantes de JavaScript como:

- `map()` => transformar datos
- `filter()` => filtrar productos
- `reduce()` => calcular totales
- `sort()` => ordenar datos
- `find()` => buscar elementos
- `some()` y `every()` => validaciones
